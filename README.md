# 🎌 Anime Industry Analysis Dashboard

## 📌 Project Overview

This project explores a comprehensive anime dataset containing information about anime titles, ratings, popularity, audience engagement, studios, genres, seasons, and release trends.

The analysis was performed using **Python** for data cleaning and exploratory data analysis (EDA), and **Power BI** for creating an interactive dashboard that provides insights into anime production, popularity, and audience preferences.

---

## 🎯 Objectives

* Analyze anime industry trends over time
* Identify the most popular and highest-rated anime
* Examine audience engagement through members and favorites
* Explore anime distribution by type, status, and source material
* Discover leading anime studios
* Investigate the relationship between anime ratings and popularity

---

## 🔍 Key Insights

* Anime production has grown significantly over the years.
* TV series dominate the anime industry.
* Manga remains one of the primary source materials for adaptations.
* High ratings do not always guarantee high popularity.
* A small number of studios contribute a substantial portion of anime productions.
* Audience engagement is concentrated among a limited number of highly popular titles.
